\section{Введение}
Данная работа посвящена исследованию задачи стабилизации перевернутого маятника с подвижной точкой опоры. Рассматриваются две стратегии управления: первая обеспечивает стабилизацию только угла отклонения маятника, вторая — полную стабилизацию (угол и положение точки опоры). В работе приведён вывод уравнений движения, анализ устойчивости с использованием критерия Рауса-Гурвица, а также результаты численного моделирования, подтверждающие теоретические выводы.

\subsection{Уравнение движения}
Уравнение движениямаятника имеет вид:
\begin{equation}
\ddot{\varphi} - \frac{g}{l} \varphi = \frac{\ddot{u}}{l}
\label{eq:dvig}
\end{equation}

\subsection{Стабилизация верхнего положения маятника}
Для стабилизации верхнего положения маятника необходимо выбрать закон управления $\ddot{u}$, который обеспечит возврат маятника в вертикальное положение при малых начальных отклонениях. Рассмотрим стратегию, в которой управление формируется как линейная комбинация угла отклонения и его угловой скорости:

\begin{equation}
\ddot{u} = -a\varphi - b\dot{\varphi}
\label{eq:uprav}
\end{equation}

Коэффициент $a$ отвечает за управление по отклонению, а коэффициент $b-$ за управление по скорости. Знак минус выбран таким образом, чтобы управление было направлено в сторону отклонения маятника: если маятник отклонился вправо ($\phi$>0), ускорение опоры должно быть отрицательным, что соответствует движению опоры вправо.

Подставим выражение для управления \eqref{eq:uprav} в уравнение маятника \eqref{eq:dvig}:

\begin{equation}
\ddot{\varphi} - \frac{g}{l} \varphi = \frac{-a\varphi - b\dot{\varphi}}{l}
\end{equation}

\begin{equation}
\ddot{\varphi} + \frac{b}{l} \dot{\varphi} + \frac{(a - g)}{l} \varphi = 0
\label{eq:lodu}
\end{equation}

Получили линейное однородное дифференциальное уравнение второго порядка с постоянными коэффициентами. Уравнение \eqref{eq:lodu} не содержит переменных $u$ и $\ddot{u}$, то есть движение опоры не влияет на устойчивость угла.

Для анализа устойчивости найдем решение уравнения в виде:
\[
\varphi(t) = Ae^{\lambda t}
\]

\[
\lambda^2A e^{\lambda t} + \frac{b}{l} \lambda Ae^{\lambda t} + \frac{(a - g)}{l} Ae^{\lambda t} = 0
\]
Сокращая на $Ae^{\lambda t}$, получим характеристическое уравнение:

\[
\lambda^2 + \frac{b}{l} \lambda + \frac{(a - g)}{l} = 0
\]

Введём обозначения:
\[
\frac{b}{l} = 2\delta, \quad \frac{(a - g)}{l} = \omega^2,
\]
где $\delta-$коэффицент затухания, а $\omega-$собственная частота системы.
\[
\lambda^2 + 2\delta\lambda + \omega^2 = 0
\]
Корни этого характеристического уравнения находятся по формуле:
\[
\lambda_{1,2} = \frac{-2\delta \pm \sqrt{4\delta^2 - 4 \cdot \omega^2}}{2}
\]

\[
\lambda_{1,2} = -\delta \pm \sqrt{\delta^2 - \omega^2}
\]

Асимптотическая устойчивость положения равновесия($\phi = 0, \dot{\phi}=0$) означает, что при любых малых начальных отклонениях решение стремится к нулю при $t \to \infty$. Для линейной системы все корни характеристического уравнения должны иметь отрицательные действительные части.

Рассмотрим возможные случаи:

\textbf{Случай 1: \( \delta^2 \geq \omega^2 \) (действительные корни)}
Подкоренное выражение неотрицательно, оба корня являются действительными числами:
\[
\lambda_1 = -\delta + \sqrt{\delta^2 - \omega^2},\qquad
\lambda_2 = -\delta - \sqrt{\delta^2 - \omega^2}.
\]

Проанализируем знаки корней:

\begin{itemize}
\item \(\lambda_2\) всегда отрицателен при \(\delta > 0\).
\item \(\lambda_1 < 0\) при \(\delta^2 - \omega^2 < \delta\).
\end{itemize}

Это неравенство выполняется всегда при \(\delta > 0\) и \(\omega^2 > 0\).

Если \(\omega^2 < 0\), то \(\delta^2 - \omega^2 > \delta\), и \(\lambda_1 > 0\) — неустойчивость.

Таким образом, в случае действительных корней для асимптотической устойчивости необходимо и достаточно, чтобы \(\delta > 0\) и \(\omega^2 > 0\)

\textbf{Случай 2: \( \delta^2 < \omega^2 \) (комплексные корни)}

В этом случае подкоренное выражение отрицательно, и корни являются комплексно-сопряженными:

\[
\lambda_{1,2} = -\delta \pm i\sqrt{\omega^2 - \delta^2}.
\]

Действительная часть равна \(-\delta\). Отрицательна при \(\delta > 0\).

\textbf{Вывод:} для асимптотической устойчивости необходимо и достаточно:

\[
\delta > 0 \quad \text{и} \quad \omega^2 > 0.
\]

Вернемся к исходным параметрам управления $a$ и $b$:

\[
\frac{b}{l} > 0 \Rightarrow b > 0,\qquad
\frac{a - g}{l} > 0 \Rightarrow a > g.
\]

Таким образом, условия асимптотической устойчивости верхнего положения маятника имеют вид:

\[
a > g, \quad b > 0.
\]

Полученные условия имеют физический смысл:

\textbf{Условие $a>g$}: управляющий момент, создаваемый перемещением опоры, должен превышать момент силы тяжести, стремящийся опрокинуть маятник. Если $a=g$, система находится на границе устойчивости — колебания не затухают. Если $a<g$, управление не может компенсировать силу тяжести, и маятник падает.

\textbf{Условие $b>0$}: введение этого параметра необходимо для затухания колебаний. При $b=0$ в системе отсутствует диссипация энергии, и колебания не затухают. При $b>0$ происходит рассеивание энергии, и система возвращается в положение равновесия.

\subsubsection{Анализ граничных случаев}
\textbf{Случай $b=0$}

Характеристическое уравнение принимает вид:
\begin{equation}
\lambda^2 + \frac{a - g}{l} = 0
\end{equation}

При $a>g$ корни чисто мнимые:
\begin{equation}
\lambda_{1,2} = \pm i\sqrt{\frac{a - g}{l}}
\end{equation}

Получили состояние равновесия центр. Система устойчива по Ляпунову, но не асимптотически.

\textbf{Случай $a=g$}
Имеем $\omega^2 = 0$, и характеристическое уравнение:
\begin{equation}
\lambda^2 + \frac{b}{l}\lambda = \lambda\left(\lambda + \frac{b}{l}\right) = 0
\end{equation}

Корни: $\lambda_1 = 0, \lambda_2=-\frac{b}{l}$. Система может совершать незатухающие колебания.

\textbf{Случай $a<g$}
Параметр $\omega^2 < 0$, тогда один из корней действительный и положительный, что соответствует росту возмущений.

\subsubsection{Недостаток стратегии управления}
При выполнении условий $a>g, b>0$ угол отклонения асимптотически стремится к нулю, но точка опоры $u(t)$ не возвращается в исходное положение. Это следует из закона управления \eqref{eq:uprav}: при при $\varphi \to 0$ и $\dot{\varphi} \to 0$ получаем $\ddot{u} \to 0$. Проинтегрируем дважды:
\begin{equation}
\dot{u}(t) \to \text{const}, \qquad u(t) \to C_1 t + C_2
\end{equation}
Таким образом, опора либо дрейфует с постоянной скоростью, либо остается в смещенном положении, отличном от нуля. На практике это означает, что жонглер, удерживающий маятник в верхнем положении, был бы вынужден неограниченно удаляться от исходной точки.

Для подтверждения полученных аналитических условий устойчивости проведем численное моделирование системы. Параметры выбраны следующими:
\begin{itemize}
    \item Длина маятника: $l = 1$ м
    \item Ускорение свободного падения: $g = 9.81$ м/с$^2$
    \item Начальное отклонение: $\varphi(0) = 0.3$ рад
    \item Начальная угловая скорость: $\dot{\varphi}(0) = 0$
    \item Начальное положение опоры: $u(0) = 0$
    \item Начальная скорость опоры: $\dot{u}(0) = 0$
\end{itemize}

\section{Анализ устойчивости полной системы}

Как было показано, управление вида
\begin{equation}
\ddot{u} = -a\varphi - b\dot{\varphi}
\label{eq:control1}
\end{equation}
обеспечивает асимптотическую устойчивость угла отклонения маятника \(\varphi(t)\) при выполнении условий \(a > g\) и \(b > 0\). Однако точка опоры \(u(t)\) при этом не возвращается в нулевое положение, а линейно растет со временем, что физически неприемлемо.

Для устранения этого недостатка необходимо расширить закон управления, добавив обратную связь по положению и скорости точки опоры. Расширенное управление имеет вид:
\begin{equation}
\boxed{\ddot{u} = a\varphi + b\dot{\varphi} + cu + d\dot{u}}
\label{eq:control_full}
\end{equation}
где \(a, b, c, d\) — параметры регулятора.

Линеаризованное уравнение движения перевернутого маятника имеет вид:
\begin{equation}
\ddot{\varphi} - \frac{g}{l}\varphi = \frac{\ddot{u}}{l}
\label{eq:pendulum}
\end{equation}

Подставляя управление \eqref{eq:control_full} в уравнение маятника \eqref{eq:pendulum}, получаем:
\begin{equation}
\ddot{\varphi} - \frac{g}{l}\varphi = \frac{1}{l}\left(a\varphi + b\dot{\varphi} + cu + d\dot{u}\right)
\end{equation}
\begin{equation}
\ddot{\varphi} - \frac{g}{l}\varphi = \frac{a}{l}\varphi + \frac{b}{l}\dot{\varphi} + \frac{c}{l}u + \frac{d}{l}\dot{u}
\end{equation}

Переносим все члены в левую часть:
\begin{equation}
\ddot{\varphi} - \frac{g}{l}\varphi - \frac{a}{l}\varphi - \frac{b}{l}\dot{\varphi} - \frac{c}{l}u - \frac{d}{l}\dot{u} = 0
\end{equation}
\begin{equation}
\ddot{\varphi} - \frac{b}{l}\dot{\varphi} - \frac{a+g}{l}\varphi - \frac{c}{l}u - \frac{d}{l}\dot{u} = 0
\end{equation}

Выражая \(\ddot{\varphi}\), получаем:
\begin{equation}
\ddot{\varphi} = \frac{b}{l}\dot{\varphi} + \frac{a+g}{l}\varphi + \frac{c}{l}u + \frac{d}{l}\dot{u}
\end{equation}

Таким образом, замкнутая система описывается двумя дифференциальными уравнениями:
\begin{equation}
\boxed{
\begin{cases}
\ddot{\varphi} = \dfrac{b}{l}\dot{\varphi} + \dfrac{a+g}{l}\varphi + \dfrac{c}{l}u + \dfrac{d}{l}\dot{u} \\[10pt]
\ddot{u} = a\varphi + b\dot{\varphi} + cu + d\dot{u}
\end{cases}
}
\label{eq:closed_system}
\end{equation}

Для анализа устойчивости ищем решение в виде экспонент:
\begin{equation}
\varphi(t) = A e^{\lambda t}, \qquad u(t) = B e^{\lambda t}
\label{eq:exp_solution}
\end{equation}

Подставляя \eqref{eq:exp_solution} в систему \eqref{eq:closed_system}, получаем:
\begin{equation}
\begin{cases}
\lambda^2 A = \dfrac{b}{l}\lambda A + \dfrac{a+g}{l} A + \dfrac{c}{l} B + \dfrac{d}{l}\lambda B \\[10pt]
\lambda^2 B = a A + b\lambda A + c B + d\lambda B
\end{cases}
\end{equation}

Переносим все члены в левую часть:
\begin{equation}
\begin{cases}
\left(\lambda^2 - \dfrac{b}{l}\lambda - \dfrac{a+g}{l}\right) A - \left(\dfrac{c}{l} + \dfrac{d}{l}\lambda\right) B = 0 \\[10pt]
(a + b\lambda) A + \left(c + d\lambda - \lambda^2\right) B = 0
\end{cases}
\label{eq:algebraic}
\end{equation}

Система \eqref{eq:algebraic} имеет ненулевое решение тогда и только тогда, когда определитель матрицы коэффициентов равен нулю:
\begin{equation}
\det \begin{pmatrix}
\lambda^2 - \dfrac{b}{l}\lambda - \dfrac{a+g}{l} & -\left(\dfrac{c}{l} + \dfrac{d}{l}\lambda\right) \\[10pt]
a + b\lambda & c + d\lambda - \lambda^2
\end{pmatrix} = 0
\label{eq:det}
\end{equation}

Вычисляем определитель:
\begin{equation}
\left(\lambda^2 - \frac{b}{l}\lambda - \frac{a+g}{l}\right)\left(c + d\lambda - \lambda^2\right) + \left(\frac{c}{l} + \frac{d}{l}\lambda\right)(a + b\lambda) = 0
\end{equation}

После раскрытия скобок и приведения подобных членов получаем характеристическое уравнение:
\begin{equation}
\boxed{\lambda^4 - \left(d + \frac{b}{l}\right)\lambda^3 - \left(c + \frac{g}{l} + \frac{a}{l}\right)\lambda^2 + \frac{g}{l}d\lambda + \frac{g}{l}c = 0}
\label{eq:char_poly}
\end{equation}

Запишем полином \eqref{eq:char_poly} в стандартной форме:
\begin{equation}
\lambda^4 + A_3\lambda^3 + A_2\lambda^2 + A_1\lambda + A_0 = 0
\end{equation}
где
\begin{equation}
A_3 = -\left(d + \frac{b}{l}\right), \quad 
A_2 = -\left(c + \frac{g}{l} + \frac{a}{l}\right), \quad 
A_1 = \frac{g}{l}d, \quad 
A_0 = \frac{g}{l}c
\end{equation}

Корни характеристического уравнения можно сделать любыми за счет выбора параметров \(a, b, c, d\). Это означает, что мы можем подобрать коэффициенты \(a, b, c, d\) (в том числе отрицательные) таким образом, чтобы все корни \(\lambda\) имели отрицательные действительные части, обеспечивая асимптотическую устойчивость системы.

\subsection{Условия устойчивости}
Для того чтобы все корни характеристического уравнения находились в левой полуплоскости (система асимптотически устойчива), параметры \(a, b, c, d\) должны удовлетворять определенным условиям. Получим эти условия, применяя критерий Рауса-Гурвица к полиному \eqref{eq:char_poly}.

\subsubsection{Необходимые условия}
Из теоремы Виета следует, что если все корни имеют отрицательные действительные части, то все коэффициенты полинома должны быть положительными:
\begin{equation}
A_3 > 0,\quad A_2 > 0,\quad A_1 > 0,\quad A_0 > 0
\end{equation}

Это дает:
\begin{align}
A_3 = -\left(d + \frac{b}{l}\right) > 0 &\quad\Rightarrow\quad d + \frac{b}{l} < 0 \quad\Rightarrow\quad d < -\frac{b}{l} \\[5pt]
A_2 = -\left(c + \frac{g}{l} + \frac{a}{l}\right) > 0 &\quad\Rightarrow\quad c + \frac{g}{l} + \frac{a}{l} < 0 \\[5pt]
A_1 = \frac{g}{l}d > 0 &\quad\Rightarrow\quad d > 0 \\[5pt]
A_0 = \frac{g}{l}c > 0 &\quad\Rightarrow\quad c > 0
\end{align}

Из условий \(d > 0\) и \(d < -\frac{b}{l}\) следует, что для устойчивости необходимо:
\begin{equation}
b < 0,\quad d > 0,\quad d < -\frac{b}{l}
\end{equation}

Аналогично, из \(c > 0\) и \(c + \frac{g}{l} + \frac{a}{l} < 0\) следует:
\begin{equation}
a < -g - l c
\end{equation}

\subsubsection{Достаточные условия устойчивости (критерий Рауса-Гурвица)}

Для определения достаточных условий устойчивости воспользуемся критерием Рауса-Гурвица. Рассмотрим характеристический полином системы:
\begin{equation}
P(\lambda) = \lambda^4 + A_3\lambda^3 + A_2\lambda^2 + A_1\lambda + A_0
\label{eq:poly}
\end{equation}
где коэффициенты для нашего случая имеют вид:
\begin{equation}
A_3 = -\left(d + \frac{b}{l}\right), \quad 
A_2 = -\left(c + \frac{g}{l} + \frac{a}{l}\right), \quad 
A_1 = \frac{g}{l}d, \quad 
A_0 = \frac{g}{l}c
\label{eq:coeffs}
\end{equation}


Для полинома четвертой степени таблица Рауса строится следующим образом. Первые две строки формируются из коэффициентов полинома:

\[
\begin{array}{c|ccc}
\lambda^4 & 1 & A_2 & A_0 \\
\lambda^3 & A_3 & A_1 & 0 \\
\end{array}
\]

Элементы третьей строки (\(\lambda^2\)) вычисляются по формулам:
\begin{align}
b_1 &= \frac{A_3 \cdot A_2 - 1 \cdot A_1}{A_3} = \frac{A_3 A_2 - A_1}{A_3} \\
b_2 &= \frac{A_3 \cdot A_0 - 1 \cdot 0}{A_3} = A_0 \\
b_3 &= 0
\end{align}

Элементы четвертой строки (\(\lambda^1\)):
\begin{align}
c_1 &= \frac{b_1 \cdot A_1 - A_3 \cdot b_2}{b_1} = \frac{b_1 A_1 - A_3 A_0}{b_1} \\
c_2 &= \frac{b_1 \cdot 0 - A_3 \cdot 0}{b_1} = 0
\end{align}

Элементы пятой строки (\(\lambda^0\)):
\begin{align}
d_1 &= \frac{c_1 \cdot b_2 - b_1 \cdot 0}{c_1} = b_2 = A_0
\end{align}

Таким образом, таблица Рауса имеет вид:
\[
\begin{array}{c|ccc}
\lambda^4 & 1 & A_2 & A_0 \\
\lambda^3 & A_3 & A_1 & 0 \\
\lambda^2 & \dfrac{A_3 A_2 - A_1}{A_3} & A_0 & 0 \\
\lambda^1 & \dfrac{(A_3 A_2 - A_1) A_1 - A_3^2 A_0}{A_3 A_2 - A_1} & 0 & 0 \\
\lambda^0 & A_0 & 0 & 0
\end{array}
\]


Согласно критерию Рауса, для асимптотической устойчивости необходимо и достаточно, чтобы все элементы первого столбца таблицы были положительными:

\begin{enumerate}
    \item \(1 > 0\) 
    
    \item \(A_3 > 0\) 
    
    \item \(\displaystyle \frac{A_3 A_2 - A_1}{A_3} > 0\).
    
    Поскольку \(A_3 > 0\) из предыдущего условия, это неравенство эквивалентно:
    \begin{equation}
    A_3 A_2 - A_1 > 0 \quad \Longleftrightarrow \quad A_3 A_2 > A_1
    \label{eq:cond2}
    \end{equation}
    
    \item \(\displaystyle \frac{(A_3 A_2 - A_1) A_1 - A_3^2 A_0}{A_3 A_2 - A_1} > 0\).
    
    Поскольку знаменатель \(A_3 A_2 - A_1 > 0\) из предыдущего условия, это неравенство эквивалентно:
    \begin{equation}
    (A_3 A_2 - A_1) A_1 - A_3^2 A_0 > 0
    \end{equation}
    \begin{equation}
    A_3 A_2 A_1 - A_1^2 - A_3^2 A_0 > 0
    \label{eq:cond3}
    \end{equation}
    
    \item \(A_0 > 0\) 
\end{enumerate}


Условия \(A_3 > 0\), \(A_2 > 0\), \(A_1 > 0\), \(A_0 > 0\) и \(A_3 A_2 > A_1\) автоматически следуют из последнего неравенства \(A_3 A_2 A_1 > A_1^2 + A_3^2 A_0\) при положительных коэффициентах. Поэтому основным достаточным условием является выполнение неравенства:
\begin{equation}
\boxed{A_3 A_2 A_1 > A_1^2 + A_3^2 A_0}
\label{eq:main_condition}
\end{equation}


Подставим выражения для коэффициентов \eqref{eq:coeffs} в условие \eqref{eq:main_condition}:
\begin{equation}
\left(-\left(d + \frac{b}{l}\right)\right) \cdot \left(-\left(c + \frac{g}{l} + \frac{a}{l}\right)\right) \cdot \left(\frac{g}{l}d\right) > \left(\frac{g}{l}d\right)^2 + \left(-\left(d + \frac{b}{l}\right)\right)^2 \cdot \left(\frac{g}{l}c\right)
\end{equation}

Произведение двух отрицательных чисел дает положительное число, а квадрат устраняет знак:
\begin{equation}
\left(d + \frac{b}{l}\right) \left(c + \frac{g}{l} + \frac{a}{l}\right) \left(\frac{g}{l}d\right) > \left(\frac{g}{l}d\right)^2 + \left(d + \frac{b}{l}\right)^2 \left(\frac{g}{l}c\right)
\end{equation}

Разделим обе части неравенства на положительную величину \(\frac{g}{l} > 0\):
\begin{equation}
d \left(d + \frac{b}{l}\right) \left(c + \frac{g}{l} + \frac{a}{l}\right) > \frac{g}{l}d^2 + c \left(d + \frac{b}{l}\right)^2
\label{eq:final_inequality}
\end{equation}

\subsubsection{Итоговые условия устойчивости}

Таким образом, для асимптотической устойчивости замкнутой системы необходимо и достаточно выполнение следующих условий:

\begin{enumerate}
    \item \(c > 0\);
    \item \(d > 0\);
    \item \(d + \dfrac{b}{l} < 0 \quad \Rightarrow \quad b < -\dfrac{d}{l}\);
    \item \(c + \dfrac{g}{l} + \dfrac{a}{l} < 0 \quad \Rightarrow \quad a < -g - lc\);
    \item выполнение неравенства:
    \begin{equation}
    \boxed{d\left(d + \frac{b}{l}\right)\left(c + \frac{g}{l} + \frac{a}{l}\right) > \frac{g}{l}d^2 + c\left(d + \frac{b}{l}\right)^2}
    \label{eq:final_stability}
    \end{equation}
\end{enumerate}

Обратим внимание, что из условий \(d > 0\) и \(d + \frac{b}{l} < 0\) следует \(b < -l d < 0\), то есть коэффициент \(b\) должен быть отрицательным. Аналогично, из \(c > 0\) и \(c + \frac{g}{l} + \frac{a}{l} < 0\) следует \(a < -g - lc < 0\), то есть коэффициент \(a\) также должен быть отрицательным.

Будем искать управление при небольшом отклонении (чтобы могли стабилизировать)

Исходное уравнение маятника с управляющим воздействием
\[
\ddot{\varphi} - \frac{g}{l} \varphi = \frac{\ddot{u}}{l} \Rightarrow \quad 
\ddot{\varphi} = \frac{\ddot{u}}{l} + \frac{g}{l} \varphi.
\]
Здесь:
\begin{itemize}
    \item \(\varphi\) — угол отклонения маятника от вертикали;
    \item \(u = u(t)\) — горизонтальное перемещение точки опоры;
    \item \(\dot{u}\) — НАПИСАТЬ;
    \item \(g\) — ускорение свободного падения;
    \item \(l\) — длина маятника.
\end{itemize}

Требуется стабилизировать маятник в верхнем неустойчивом положении 
(\(\varphi = 0,\; \dot{\varphi} = 0\)). Для этого необходимо подобрать 
такое управление \(\ddot{u}\), чтобы замкнутая система обладала 
устойчивым состоянием равновесия в точке \((\varphi,\; \dot{\varphi}) = (0, 0)\).

Желаемое поведение маятника зададим уравнением затухающего гармонического осциллятора:
\[
\ddot{\varphi} + 2\delta \dot{\varphi} + w^2 \varphi = 0.
\]
подставим туда фи две точки
\[
\frac{\ddot{u}}{l} + \frac{g}{l} \varphi + 2\delta \dot{\varphi} + w^2 \varphi = 0.
\]
Получили уравнение для управляющей переменной u:
\[
\ddot{u} = -(g + l w^2) \varphi - 2\delta l \dot{\varphi}.
\]

Это уравнение представляет собой линейную обратную связь по состоянию:
\begin{equation}
\ddot{u} = -a\varphi - b\dot{\varphi}, \label{eq:state_feedback}
\end{equation}
где коэффициенты обратной связи:
\[
a = g + l\omega^2, \qquad b = 2\delta l.
\]

Их физический смысл ХЗ НАДО ЛИ:

\begin{itemize}
    \item \(a\) — угловая коррекция (приведение). Слагаемое \(-a\varphi\) означает, 
    что точка опоры ускоряется в ту же сторону, куда отклонился маятник. 
    Чем больше отклонение, тем сильнее ускорение.
    
    \item \(b\) — скоростная коррекция (одерживание). Слагаемое \(-b\dot{\varphi}\) 
    увеличивает управление, если маятник движется в сторону отклонения 
    (\(\varphi\) и \(\dot{\varphi}\) одного знака), и уменьшает, если маятник 
    возвращается (\(\varphi\) и \(\dot{\varphi}\) разных знаков), предотвращая 
    ``проскакивание'' вертикали.
\end{itemize}











\subsection{Замкнутая система и анализ устойчивости}

Подставим закон управления в уравнение движения. Для этого выразим $\ddot{u}$ из закона управления и подставим в линеаризованное уравнение маятника:

\[
\ddot{\varphi} - \frac{g}{l} \varphi = \frac{1}{l} (-a\varphi - b\dot{\varphi}).
\]

Перенесем все слагаемые в левую часть:

\[
\ddot{\varphi} + \frac{b}{l} \dot{\varphi} + \left( \frac{a}{l} - \frac{g}{l} \right) \varphi = 0.
\]

С учетом выражений $a = g + l\omega^2$ и $b = 2\delta l$ получаем:

\[
\frac{b}{l} = 2\delta, \qquad \frac{a}{l} - \frac{g}{l} = \frac{g + l\omega^2}{l} - \frac{g}{l} = \omega^2.
\]

Таким образом, уравнение замкнутой системы принимает вид:

\[
\ddot{\varphi} + 2\delta \dot{\varphi} + \omega^2 \varphi = 0,
\]

что полностью совпадает с эталонным уравнением затухающего осциллятора.

Характеристический полином замкнутой системы:

\[
\lambda^2 + 2\delta \lambda + \omega^2 = 0.
\]

Для асимптотической устойчивости (чтобы маятник возвращался в вертикальное положение) необходимо и достаточно, чтобы все корни характеристического уравнения имели отрицательные вещественные части. Для уравнения второго порядка критерий Гурвица дает условия (МОЖЕТ ЛУЧШЕ ЗАПИСАТЬ КАК-ТО, А ТО НЕПОНЯТНО, ЧТО ЗА КРИТЕРИЙ):

\[
\delta > 0, \qquad \omega^2 > 0.
\]

В терминах коэффициентов обратной связи это означает:

\[
b = 2\delta l > 0 \quad \Rightarrow \quad b > 0,
\]
\[
a = g + l\omega^2 > g \quad \Rightarrow \quad a > g.
\]

Таким образом, при $a > g$ и $b > 0$ верхнее положение маятника становится асимптотически устойчивым.

\subsection{ Ограничения предложенной стратегии}

Полученная стратегия управления $\ddot{u} = -a\varphi - b\dot{\varphi}$ успешно стабилизирует угол $\varphi$ и угловую скорость $\dot{\varphi}$. Однако она не решает задачу стабилизации положения точки опоры $u$.

Действительно, проанализируем, что происходит с переменной $u$. Из закона управления при $\varphi \to 0$ и $\dot{\varphi} \to 0$ получаем $\ddot{u} \to 0$. Интегрируя, находим:

\[
\dot{u}(t) = \dot{u}(0) = \text{const}, \qquad u(t) = u(0) + \dot{u}(0) \cdot t.
\]

Это означает, что:
\begin{itemize}
    \item если $\dot{u}(0) = 0$, то точка опоры остается на месте, но на том смещении $u(0)$, которое было в начальный момент;
    \item если $\dot{u}(0) \neq 0$, то точка опоры движется с постоянной скоростью и неограниченно удаляется.
\end{itemize}

Следовательно, при данной стратегии управления положение точки опоры не стабилизируется: она либо остается смещенной на величину начального отклонения, либо движется равномерно. Для практической реализации необходимо, чтобы точка опоры возвращалась в исходное положение, что требует расширения закона управления.

\section{Расширенная стратегия управления для полной стабилизации}

Для решения задачи одновременной стабилизации верхнего положения маятника и точки опоры необходимо ввести обратную связь не только по $\varphi$ и $\dot{\varphi}$, но и по $u$ и $\dot{u}$. Расширенный закон управления имеет вид:

\[
\ddot{u} = a\varphi + b\dot{\varphi} + c u + d \dot{u}.
\]



\subsection{Замкнутая система}

Подставим закон управления в линеаризованное уравнение движения маятника:

\[
\ddot{\varphi} - \frac{g}{l} \varphi = \frac{1}{l} \left( a\varphi + b\dot{\varphi} + c u + d \dot{u} \right).
\]

Перенесем все слагаемые в левую часть:

\[
\ddot{\varphi} - \frac{g}{l} \varphi - \frac{a}{l} \varphi - \frac{b}{l} \dot{\varphi} - \frac{c}{l} u - \frac{d}{l} \dot{u} = 0.
\]

Сгруппируем:

\[
\ddot{\varphi} - \frac{b}{l} \dot{\varphi} - \left( \frac{g}{l} + \frac{a}{l} \right) \varphi - \frac{c}{l} u - \frac{d}{l} \dot{u} = 0
\]

Получаем замкнутую систему: (ПОЧЕМУ ЗАМКНУТУЮ И ЧТО ЭТО ЗНАЧИТ)
\[
\begin{cases}
\ddot{\varphi} - \dfrac{g}{l} \varphi = \dfrac{\ddot{u}}{l} \\[10pt]
\ddot{\varphi} - \dfrac{b}{l} \dot{\varphi} - \left( \dfrac{g}{l} + \dfrac{a}{l} \right) \varphi - \dfrac{c}{l} u - \dfrac{d}{l} \dot{u} = 0
\end{cases}
\]

\subsection{Переход к системе первого порядка}

Введем переменные состояния:

\[
x_1 = \varphi, \quad x_2 = \dot{\varphi}, \quad x_3 = u, \quad x_4 = \dot{u}.
\]

Тогда:

\[
\begin{cases}
\dot{x}_1 = x_2, \\[1em]
\dot{x}_2 = \ddot{\varphi} = \dfrac{b}{l} x_2 + \left( \dfrac{g}{l} + \dfrac{a}{l} \right) x_1 + \dfrac{c}{l} x_3 + \dfrac{d}{l} x_4, \\[1em]
\dot{x}_3 = x_4, \\[1em]
\dot{x}_4 = \ddot{u} = a x_1 + b x_2 + c x_3 + d x_4.
\end{cases}
\]

\subsection{Матричная форма}

\[
\frac{d}{dt}
\begin{pmatrix}
x_1 \\ x_2 \\ x_3 \\ x_4
\end{pmatrix}
=
\begin{pmatrix}
0 & 1 & 0 & 0 \\[1em]
\dfrac{g + a}{l} & \dfrac{b}{l} & \dfrac{c}{l} & \dfrac{d}{l} \\[1em]
0 & 0 & 0 & 1 \\[1em]
a & b & c & d
\end{pmatrix}
\begin{pmatrix}
x_1 \\ x_2 \\ x_3 \\ x_4
\end{pmatrix}. \qquad (5.10)
\]

\subsection{Характеристическое уравнение}

Характеристический полином находится из $\det(A - \lambda E) = 0$:

\[
\det
\begin{pmatrix}
-\lambda & 1 & 0 & 0 \\[1em]
\dfrac{g + a}{l} & \dfrac{b}{l} - \lambda & \dfrac{c}{l} & \dfrac{d}{l} \\[1em]
0 & 0 & -\lambda & 1 \\[1em]
a & b & c & d - \lambda
\end{pmatrix} = 0. \qquad (5.11)
\]

Вычислим определитель. Разложим по первому столбцу. Элементы первого столбца:
\begin{itemize}
    \item $A_{11} = -\lambda$, минор $M_{11}$,
    \item $A_{21} = \dfrac{g + a}{l}$, минор $M_{21}$,
    \item $A_{31} = 0$,
    \item $A_{41} = a$, минор $M_{41}$.
\end{itemize}

С учетом знаков $(-1)^{i+1}$:

\[
\det = (-\lambda) \cdot (-1)^{2} M_{11} + \frac{g + a}{l} \cdot (-1)^{3} M_{21} + 0 + a \cdot (-1)^{5} M_{41}.
\]

Поскольку $(-1)^{2} = 1$, $(-1)^{3} = -1$, $(-1)^{5} = -1$, получаем:

\[
\det = (-\lambda) M_{11} - \frac{g + a}{l} M_{21} - a M_{41}. \qquad (5.12)
\]

\subsection{Вычисление миноров}

\textbf{Минор $M_{21}$} (вычеркиваем 2-ю строку и 1-й столбец):

\[
M_{21} = \det
\begin{pmatrix}
1 & 0 & 0 \\[1em]
0 & -\lambda & 1 \\[1em]
b & c & d - \lambda
\end{pmatrix}.
\]

Разложим по первой строке:
\[
M_{21} = 1 \cdot \det \begin{pmatrix} -\lambda & 1 \\ c & d - \lambda \end{pmatrix} = (-\lambda)(d - \lambda) - 1 \cdot c = \lambda^2 - d\lambda - c.
\]

\textbf{Минор $M_{41}$} (вычеркиваем 4-ю строку и 1-й столбец):

\[
M_{41} = \det
\begin{pmatrix}
1 & 0 & 0 \\[1em]
\dfrac{b}{l} - \lambda & \dfrac{c}{l} & \dfrac{d}{l} \\[1em]
0 & -\lambda & 1
\end{pmatrix}.
\]

Разложим по первой строке:
\[
M_{41} = 1 \cdot \det \begin{pmatrix} \dfrac{c}{l} & \dfrac{d}{l} \\ -\lambda & 1 \end{pmatrix} = \frac{c}{l} \cdot 1 - \frac{d}{l} \cdot (-\lambda) = \frac{c}{l} + \frac{d\lambda}{l}.
\]

\textbf{Минор $M_{11}$} (вычеркиваем 1-ю строку и 1-й столбец):

\[
M_{11} = \det
\begin{pmatrix}
\dfrac{b}{l} - \lambda & \dfrac{c}{l} & \dfrac{d}{l} \\[1em]
0 & -\lambda & 1 \\[1em]
b & c & d - \lambda
\end{pmatrix}.
\]

Разложим по первому столбцу:
\[
M_{11} = \left( \frac{b}{l} - \lambda \right) \cdot \det \begin{pmatrix} -\lambda & 1 \\ c & d - \lambda \end{pmatrix} - 0 + b \cdot \det \begin{pmatrix} \dfrac{c}{l} & \dfrac{d}{l} \\ -\lambda & 1 \end{pmatrix}.
\]

Первый минор:
\[
\det \begin{pmatrix} -\lambda & 1 \\ c & d - \lambda \end{pmatrix} = \lambda^2 - d\lambda - c.
\]

Второй минор:
\[
\det \begin{pmatrix} \dfrac{c}{l} & \dfrac{d}{l} \\ -\lambda & 1 \end{pmatrix} = \frac{c}{l} + \frac{d\lambda}{l}.
\]

Следовательно:
\[
M_{11} = \left( \frac{b}{l} - \lambda \right) (\lambda^2 - d\lambda - c) + b \left( \frac{c}{l} + \frac{d\lambda}{l} \right).
\]

\subsection{Раскрытие $M_{11}$}

\[
M_{11} = \frac{b}{l} (\lambda^2 - d\lambda - c) - \lambda (\lambda^2 - d\lambda - c) + \frac{bc}{l} + \frac{bd\lambda}{l}.
\]

\[
M_{11} = \frac{b}{l}\lambda^2 - \frac{bd}{l}\lambda - \frac{bc}{l} - \lambda^3 + d\lambda^2 + c\lambda + \frac{bc}{l} + \frac{bd\lambda}{l}.
\]

Слагаемые $-\frac{bc}{l}$ и $+\frac{bc}{l}$ сокращаются. Слагаемые $-\frac{bd}{l}\lambda$ и $+\frac{bd\lambda}{l}$ также сокращаются. Остается:

\[
M_{11} = -\lambda^3 + \left( \frac{b}{l} + d \right) \lambda^2 + c\lambda.
\]

\subsection{Подстановка в определитель}

Подставим $M_{11}$, $M_{21}$ и $M_{41}$ в (5.12):

\[
\det = (-\lambda) \left[ -\lambda^3 + \left( \frac{b}{l} + d \right) \lambda^2 + c\lambda \right] - \frac{g + a}{l} (\lambda^2 - d\lambda - c) - a \left( \frac{c}{l} + \frac{d\lambda}{l} \right).
\]

Раскроем скобки:

\[
\det = \lambda^4 - \left( \frac{b}{l} + d \right) \lambda^3 - c\lambda^2 - \frac{g + a}{l} \lambda^2 + \frac{d(g + a)}{l} \lambda + \frac{c(g + a)}{l} - \frac{ac}{l} - \frac{ad\lambda}{l}.
\]

\subsection{Приведение подобных}

Сгруппируем члены при $\lambda^2$:
\[
- c\lambda^2 - \frac{g + a}{l} \lambda^2 = -\left( c + \frac{g + a}{l} \right) \lambda^2.
\]

Сгруппируем члены при $\lambda$:
\[
\frac{d(g + a)}{l} \lambda - \frac{ad}{l} \lambda = \frac{dg + da - ad}{l} \lambda = \frac{dg}{l} \lambda.
\]

Сгруппируем свободные члены:
\[
\frac{c(g + a)}{l} - \frac{ac}{l} = \frac{cg + ca - ac}{l} = \frac{cg}{l}.
\]

\subsection{Итоговый характеристический полином}

Таким образом, получаем:

\[
\det = \lambda^4 - \left( \frac{b}{l} + d \right) \lambda^3 - \left( c + \frac{g + a}{l} \right) \lambda^2 + \frac{dg}{l} \lambda + \frac{cg}{l} = 0.
\]

Перепишем в виде, принятом в методичке:

\[
\boxed{\lambda^4 + \left( -d - \frac{b}{l} \right) \lambda^3 + \left( -c - \frac{g}{l} - \frac{a}{l} \right) \lambda^2 + \frac{dg}{l} \lambda + \frac{cg}{l} = 0.} \qquad (5.8)
\]

\subsection{Анализ возможности стабилизации}

Корни уравнения (5.8) за счет выбора параметров $a, b, c, d$ можно сделать любыми. Это следует из того, что:
\begin{itemize}
    \item свободный член $\dfrac{cg}{l}$ можно сделать любым за счет выбора $c$;
    \item коэффициент при $\lambda$ — $\dfrac{dg}{l}$ — за счет выбора $d$;
    \item коэффициент при $\lambda^2$ — $\left( -c - \dfrac{g}{l} - \dfrac{a}{l} \right)$ — за счет выбора $a$ (при фиксированном $c$);
    \item коэффициент при $\lambda^3$ — $\left( -d - \dfrac{b}{l} \right)$ — за счет выбора $b$ (при фиксированном $d$).
\end{itemize}

Таким образом, стратегия вида (5.7) за счет надлежащего выбора значений параметров $a, b, c, d$, при которых все корни уравнения (5.8) лежат слева от мнимой оси, обеспечивает асимптотическую устойчивость состояния равновесия $(\varphi, \dot{\varphi}, u, \dot{u}) = (0,0,0,0)$. Это означает, что маятник возвращается в вертикальное положение, а точка опоры — в исходное положение и останавливается.

\subsection{Общий вывод}
Для полной стабилизации (одновременного возврата угла и точки опоры в нуль) необходимо использовать управление вида $\ddot{u} = a\varphi + b\dot{\varphi} + cu + d\dot{u}$ с параметрами, удовлетворяющими условиям:
\begin{itemize}
    \item $c > 0$, $d > 0$;
    \item $a < -g - lc$ (коэффициент $a$ должен быть отрицательным и достаточно большим по модулю);
    \item $b < -\dfrac{d}{l}$ (коэффициент $b$ должен быть отрицательным и достаточно большим по модулю).
\end{itemize}
При этом выбор $c$ и $d$ должен быть таким, чтобы система не выходила на границу устойчивости: слишком малые $d$ или слишком большие $c$ могут привести к нарастанию колебаний.
